## Specification

This specification is established so that the project "Ecchi Fanservice Anime List with Visible Nipple Exposure" can continue smoothly in the future.

It defines the workflow for creating the table, the inclusion criteria for anime, and the rules to follow.

The "Ecchi Fanservice Anime List with Visible Nipple Exposure" began around November 2021, and its structure has changed many times. This specification was written with current global trends in mind and may be revised from time to time.

**This specification describes the ideal situation and serves as a reference for future work. In practice, the existing data still contains many parts that need correction.**


### Anime Media Scope

This list covers anime broadcast since 1917, although the earliest works I can find are from the 1970s, plus some Osamu Tezuka animations from the 1960s.

Items included in this list:
- TV anime, meaning non-adult mainstream anime
- OVA/OAD/bonus episodes
- Anime adapted from games or manga
- Short anime, such as anime produced by Suiseisha
- Anime films

Items not included in this list:
- Adult anime, meaning hentai/uraban
- In-game animation
- Doujin animation, such as Touhou Project
- Non-Japanese animation
- Experimental works from art festivals
- Live-action dramas


### Criteria for Nipple Exposure and Ecchi Fanservice

Although this list is called an "anime with nipple exposure" list and mentions ecchi fanservice many times, the emphasis is on anime that actually contains nipple exposure. Being an ecchi fanservice anime does not necessarily mean it has nipple exposure.

There are far too many anime with fanservice shots. If panty shots and swimsuits were counted too, the list would be endless, so visible nipple exposure is the primary requirement.

Anime with nipple exposure in only one episode or one short segment can still be included in this list.

In short, an anime must contain actual nipple exposure to be included. Ecchi fanservice elements are only a bonus.

These count as nipple exposure:
- The character must be female. MTF characters are also accepted. Species does not matter, but the character must at least be humanoid.
- Nipple exposure means a shot that clearly depicts nipples on breasts.
- Nipple exposure in dark lighting, such as Unbreakable Machine-Doll
- "Poking through," where clothing clearly reveals the shape of the nipples
- Visible areolae, such as Anna Nishikinomiya in Shimoneta S01E04
- Nipples visible through wet clothing, such as Yukina Himeragi in Strike the Blood S01E02
- A nude body with the outline of nipples drawn, such as Asuna in Sword Art Online The Movie: Ordinal Scale
- Very casually drawn nipples, such as Akiho Kosaka in My Girlfriend is Shobitch OVA01

These do not count as nipple exposure:
- Panty shots
- Exposed underwear
- Exposed buttocks
- Exposed legs
- Exposed armpits. In short, personal fetish areas other than nipples do not count as nipple exposure.
- Bathing or hot spring scenes
- Dirty jokes
- Sexually suggestive gestures
- Real existing artwork rather than the character themselves exposing nipples
- Breast shots obscured by censor light or foreign objects
- Sexual intercourse positions where only a rough body outline is depicted
- Large areas of nudity without nipples being drawn, such as Shiragiku Ono in The Cafe Terrace and Its Goddesses S01E01
- Full-body nudity without nipples being drawn, such as the "super healthy space" in No Game No Life or the censor-light transformation scenes in Sailor Moon
- Grotesque content
- BL scenes
- Male nudity

If you find an excellent ecchi fanservice anime that cannot be included in this list because it has no nipple exposure, it can be recycled into the "Useful Ecchi Fanservice Anime List Without Nipple Exposure."

### Anime Video Sources

For currently airing seasonal anime, wait until all episodes have aired before taking screenshots.

Blu-ray sources are preferred. They are usually released several months after the anime airs.

The first priority is to use P2P software to find and download 10-bit 1080p BDRip (Blu-ray) sources from Nyaa. The BD version is usually the uncensored version.

If possible, buy official BD releases and rip them yourself.

The next option is streaming (WEB-DL) sources, such as Bahamut Anime Crazy, Bilibili, or other streaming sites. If you use streaming sources in order to keep up with a new seasonal anime, it is recommended to replace them with BD sources later.

Do not use TV version (HDTV) sources. They are often heavily censored.

### Anime Table Fields

Anime data is recorded in JSON, as described below.

All anime data will eventually be rendered into one HTML table.

In the table, horizontal cells are called rows. Each row represents one anime.

Vertical cells are called columns. There are four columns:

1. Anime title, which must include translations in three languages
2. Broadcast year, which must be specific down to year, month, and day
3. Nipple exposure episode numbers and preview screenshots. Screenshots must include episode and character information, and must include translations in three languages.
4. Commentary and snark. This must include translations in three languages. It may be left blank.

If an anime has many seasons, each season and OVA should be listed as a separate row. If there are multiple OVAs, it is also recommended to list them separately.

This is not an anime encyclopedia. Only list seasons that contain nipple exposure. Even ecchi fanservice anime do not necessarily have nipple exposure in every season.

For example:
```text
- IS: Infinite Stratos
- IS: Infinite Stratos Season 2
- IS: Infinite Stratos
- IS: Infinite Stratos OVA Encore: A Sextet Yearning for Love
- IS: Infinite Stratos OVA World Purge-hen
```

There is no need to specifically write "Season 1" for the first season. If a second season has its own title, use that title first instead of calling it "Season 2."

For example:
```text
- Peter Grill and the Philosopher's Time
- Peter Grill and the Philosopher's Time: Super Extra
```

For Chinese translations of anime titles, use the Taiwanese title. If there is no Taiwanese release, use the Chinese mainland title or the commonly used online title.

For English translations of anime titles, prioritize Japanese romanization without macrons or long-vowel marks. Use translated titles when appropriate.

Anime broadcast dates must be specific down to month and day. Use Japan time as the basis. This is not the BD release date.


### Taking Preview Screenshots of Nipple Exposure Episodes

Due to layout constraints, each anime may have at most three preview images. Screenshots must not include subtitles.

For anime after 2000, use 10-bit 1080p BDRip sources as much as possible.

In principle, do not take screenshots from OP or ED sequences. Ideally, the three screenshots should come from different episodes and different characters.

Although other shots may often be more visually appealing, prioritize shots with actual nipple exposure and the character's face, so that they match the standard of this list.

Prioritize main characters. Prefer shots that show the nipple exposure area and the character's face. The next best option is a shot with nipple exposure but only a breast-focused angle. Depending on the fetish emphasis of the work, screenshots of buttocks or genital areas may be taken at your discretion.

If there is truly only one nipple exposure shot, use the remaining two images to supplement it with very ecchi scenes that do not contain nipple exposure. If there are truly no other very ecchi scenes, then one image is enough.

Save screenshots as WEBP. If the original file is PNG, convert it to WEBP with 90% quality.

Filename format:
```text
Anime-Title_Season_Episode_Character-Name.extension
```

Replace spaces between anime and character names with hyphens. Filenames may contain only English letters and numbers, and must not contain special symbols forbidden by Linux or Windows filesystems.

Use `S` for season. If the number is less than 10, add a leading zero.

Use `E` for episode. If the number is less than 10, add a leading zero.

Use `SP` for specials. If the number is less than 10, add a leading zero.

Use `Movie` for theatrical films that have no special title.

Use the original names for `OVA`, `OAD`, and other special labels. If the number is less than 10, add a leading zero.

For example:
```text
IS-Infinite-Stratos_S01E06_Charlotte-Dunois.webp
```

If a screenshot contains multiple characters, separate them with underscores. Only female character names need to be marked.

For example:
```text
IS-Infinite-Stratos_S01E08_Charlotte-Dunois_Laura-Bodewig.webp
```

Filenames should use Japanese romanization without macrons or long-vowel marks. Write names in normal order and do not reverse them into the English surname-first format.

If all three images are of the same character and the same episode, add `_1`, `_2`, and `_3` to the end of the image filenames.

Image caption strings must include three languages and be rendered on the images through HTML `<title>` and `<alt>` attributes.

For readability on mobile, the captions are also rendered below the images as `p` elements.


### JSON File Structure

JSON usually does not allow `//` comments. Although the JSONC standard exists, Hugo compilation will fail. Therefore, comments are instead represented by a JSON value that does not render to the actual webpage. The `//` comments below are only for explanation.

Split the data by decade, such as 1970-1979, 1980-1989, and so on. The reason is that putting all years on one webpage would make the page too large, while splitting the content by individual year would make each page too small. Decade-based splitting also makes it easier to observe changes in art styles across different eras.

Split the data into 2 JSON files and multiple webpages.
```bash
ecchi-fanservice-anime-list.json
uncensored-ecchi-anime-list.json
```

The outermost layer of each JSON file is an array.
```json
[

]
```

The array contains multiple objects, and each object represents one anime.
```json
[
    {
    // Anime 1
    },
    {
    // Anime 2
    }
]
```
